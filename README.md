# Система "Больница"

## Описание проекта
Данное программное обеспечение предназначено для автоматизации процессов в медицинском учреждении. Система позволяет врачам вести историю болезней пациентов, назначать лечение и контролировать его выполнение.

## Основная функциональность
1.  **Врач** может:
    *   Определить диагноз пациенту.
    *   Сделать назначение (процедуры, лекарства, операции).
    *   Выполнить любое назначение (включая операции).
    *   Выписать пациента, зафиксировав окончательный диагноз.

2.  **Медсестра** может:
    *   Выполнить назначения Врача (процедуры, лекарства).

3.  **Пациент** имеет статус нахождения в больнице и при выписке получает окончательный диагноз.
